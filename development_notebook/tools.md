# Cheat Sheet

## tmux

### Session
tmux list-sessions
tmux attach-session -t \<session name\>

### Windows
`Ctrl + b` then \<arrow-key\>
    ? Change window

`Ctrl + b` then `%`
    ? Split horizontal

`Ctrl + b` then `"`
    ? Split vertical

`Ctrl + b` then `:resize-pane -[UDLR] N`
    ? Resize window by `N` to [Up, Down, Left, Right]
